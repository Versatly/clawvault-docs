# Task Views

ClawVault tasks are markdown files with frontmatter. Combined with Obsidian's Bases feature (1.12+), you get database-like views over your tasks.

## Included Base Files

The plugin can generate `.base` files for common task views. Use the command palette: **ClawVault: Generate Task Bases**.

### All Tasks

Shows every task with columns for status, priority, project, and dates.

```yaml
# tasks-all.base
filter:
  - file.inFolder("tasks")
properties:
  - { name: status, type: text }
  - { name: priority, type: text }
  - { name: project, type: text }
  - { name: created, type: date }
  - { name: updated, type: date }
views:
  - type: table
    name: All Tasks
    groupBy: status
```
