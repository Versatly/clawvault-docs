The `canvas` command generates visual dashboards as Obsidian JSON Canvas files. Choose from built-in templates or filter by project, owner, or scope.

## Usage

```bash
clawvault canvas [options]
```

### Options

| Flag | Description | Default |
|------|-------------|---------|
| `-v, --vault <path>` | Vault path | Nearest vault |
| `--output <path>` | Output canvas file path | `dashboard.canvas` |

## Examples

```bash
# Default dashboard
clawvault canvas

# Custom output path
clawvault canvas --output brain-dashboard.canvas

# Specific vault
clawvault canvas -v ~/my-vault
```

## Details

Canvas files follow the [JSON Canvas specification](https://jsoncanvas.org) and open directly in Obsidian. File nodes are clickable â€” task cards link to their markdown files.

The `brain` template reads from the memory graph index (run `clawvault graph build` first for best results). The `project-board` template reads from `tasks/` and `backlog/` directories.
